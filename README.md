# Boolean Algebra

This package provides a parser for boolean algebra.

Supported symbols are:
- '0' (false)
- '1' (true)
- '&' (and)
- '|' (or)
- '!' (not)
- '=' (equal, biimplication)
- '%' (xor)
- '>' (implication, a > b ≡ !a | b)
- '<' (reverse implication, a < b ≡ a | !b)

Literals must start with a letter.
